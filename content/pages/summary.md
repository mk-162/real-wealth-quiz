One long scrolling page with seven sections: aspiration echo → emotional-state intro → spotlight compound flag → considered list with inline charts → things we didn't ask → bridge paragraph → segment CTA → FCA footer.

The aspiration echo blends keywords from the user's Q2.4 answer with a template tied to their Q1.2 wealth-definition choice. Verbatim text on the page is generated client-side from the persisted session and is not stored — the page is refreshed from source answers each render.

The emotional-state intro selects one of six variants by priority: urgency this-week → advised-but-looking → money stress → retirement unease → S1 Early Accumulator → S9 HNW. Fallback renders the neutral intro.

The spotlight compound flag (Section 3) renders only when one of the four v1.0 flags triggers. It uses the same teal gradient treatment as the enhanced CTA.

The silent-gap section (Section 5) renders only when two or more triggers match — the point is surprise, not volume.

The bridge paragraph (Section 6) hides for Tier C users.

The enhanced CTA treatment (teal gradient, italic display headline) applies only when `current_adviser = yes_but_looking`.
